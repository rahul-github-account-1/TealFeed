## Tech Stack

- Javascript, React.js, CSS, HTML
- Libraries Used: Prism, prism-react-renderer

## Requirements

- Node.js

## Setup Instructions
### Install Dependencies and Run Code:
Locate to project folder and enter below command:

```bash
npm install
npm run dev
```
### Now open URL: http://localhost:5173/
